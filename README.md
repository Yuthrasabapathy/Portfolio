# Portfolio Project

## Overview

This project is a modern web application built using React and Vite. It focuses on performance, clean UI design, and scalable component structure. The application uses TypeScript for better maintainability and Tailwind CSS for efficient styling.

## Tech Stack

The project is developed using the following technologies:

* React
* Vite
* TypeScript
* Tailwind CSS
* shadcn/ui

## Getting Started

To run this project locally, make sure you have **Node.js and npm** installed.

### Installation Steps

Clone the repository:

```
git clone <https://github.com/Yuthrasabapathy/Portfolio.git>
```

Navigate into the project folder:

```
cd <yuthrasabapathy-portfolio>
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

The application will start with live reloading enabled.
